## How to Use

### For New Agents
1. **First**: Share [Copilot_Local/README.md](Copilot_Local/README.md)
2. **Reference**: Keep [RULES.md](Copilot_Local/RULES.md) handy
3. **Details**: Point to [GUIDELINES.md](Copilot_Local/GUIDELINES.md) when needed
4. **Maintenance**: Reference [REPOSITORY_MAINTENANCE.md](Copilot_Local/REPOSITORY_MAINTENANCE.md)

### For Answering Questions
- "Is this allowed?" → Point to [RULES.md](Copilot_Local/RULES.md) tables
- "Where does this go?" → Point to [GUIDELINES.md](Copilot_Local/GUIDELINES.md) section
- "Why that rule?" → Share [GUIDELINES.md](Copilot_Local/GUIDELINES.md) explanation
- "How to maintain?" → Point to [REPOSITORY_MAINTENANCE.md](Copilot_Local/REPOSITORY_MAINTENANCE.md)

### For Enforcing Rules
1. Reference specific rule in [RULES.md](Copilot_Local/RULES.md)
2. Show example from [GUIDELINES.md](Copilot_Local/GUIDELINES.md)
3. Use checklist or process

### For Making Changes
1. Update rule in [RULES.md](Copilot_Local/RULES.md)
2. Update explanation in [GUIDELINES.md](Copilot_Local/GUIDELINES.md)
3. Add example if helpful
4. Reference in [README.md](Copilot_Local/README.md) if needed
