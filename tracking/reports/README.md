# Reports Directory

This directory contains generated reports comparing the plan (in this repo) with actual implementation progress (from Jarvis repo).

## Report Types

### `progress-report.md`
Overall progress against the roadmap:
- Current phase
- Completed milestones
- Pending tasks
- Timeline adherence

### `deviation-report.md`
Tracks deviations from the original plan:
- Scope changes
- Technology changes
- Timeline changes
- Reasons and impact

### `component-status.md`
Status of each component:
- Implementation status (not started, in progress, complete)
- Test coverage
- Known issues
- Dependencies

### `weekly-summary.md`
Weekly development summary:
- What was completed
- What's in progress
- Blockers
- Next week's focus

## Generation

These reports can be:
1. Auto-generated by your tracking program
2. Manually created based on snapshot data
3. Hybrid (auto-generated with manual annotations)
