# Documentation

This directory contains additional documentation for the Jarvis cluster management system.

## Structure

### `/architecture`
Detailed architecture documentation, diagrams, and system design docs.

- Component interactions
- Data flow diagrams
- System architecture diagrams
- Network topology

### `/decisions`
Architectural Decision Records (ADRs) documenting key technical decisions.

Format: `ADR-NNNN-title.md`

Example:
- ADR-0001-choose-programming-language.md
- ADR-0002-state-store-selection.md

### `/guides`
How-to guides, tutorials, and operational documentation.

- Setup guides
- Deployment guides
- Troubleshooting guides
- Best practices

### `/examples`
Example configurations, use cases, and sample code.

- Sample task definitions
- Configuration examples
- API usage examples
- Integration examples

## Contributing to Documentation

1. Keep documentation close to the code when possible
2. Use Markdown format for all text documents
3. Include diagrams using Mermaid or ASCII art
4. Update the relevant documentation when making changes
5. Review [REQUIREMENTS.md](../REQUIREMENTS.md) for documentation standards
