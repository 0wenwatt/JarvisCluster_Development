# Configuration

This directory contains configuration files for the JarvisCluster_Development repository.

## Files

### `tracking-config.yaml`
Configuration for automated tracking integration with the Jarvis implementation repo.

### `gitignore-suggestions.txt`
Suggestions for files to ignore (tracking data snapshots can be large).

## Usage

Edit these files to customize:
- Which metrics to track
- Update frequency
- Report formats
- Alert thresholds
