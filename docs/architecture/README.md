# Architecture Documentation

This directory contains detailed architecture documentation for Jarvis.

## Documents

- **system-overview.md**: High-level system architecture
- **component-design.md**: Detailed component designs
- **data-flow.md**: Data flow and communication patterns
- **deployment-architecture.md**: Deployment topologies

## Diagrams

All diagrams should be created using:
- Mermaid (preferred for markdown embedding)
- ASCII art (for simple diagrams)
- Draw.io/Excalidraw (export as PNG/SVG)

Store diagram source files alongside their rendered versions.
